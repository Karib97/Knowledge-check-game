This game is called knowledge check. 
It gives you a choice of 2 categories: State Capitals or National Animals. I chose this game because when im on tiktok i like to play the random quiz games that show up on my feed. So I created a little quiz to test your general knowledge.
Here is the link the link to to the game to play: https://karib97.github.io/Knowledge-check-game/
To play: Select a category and answer the questions. To win you must score at least 4 points. 

To build this game i used HTML, CSS, and JavaScript
